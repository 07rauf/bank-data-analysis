# 📊 Bank-Data-Analysis

## 🛠 İstifadə olunan alətlər
Pandas,NumPy,Seaborn,Matplotlib,Scikit-Learn

## 📁 Layihənin təsviri
Bu layihə çərçivəsində verilən dataset üzərində **Exploratory Data Analysis (EDA)** həyata keçirilmişdir.Məlumatlar Python kitabxanaları vasitəsilə yüklənmiş,təmizlənmiş və analiz edilmişdir.Məqsəd dataset-in strukturunu daha yaxşı anlamaq, əsas statistik göstəriciləri əldə etmək və fərqli tendensiyaları aşkar etmək olmuşdur.

## 📈 Əldə olunan nəticələr
- Dataset üzərində aparılan geniş analizlər nəticəsində dəyişənlər arasında əhəmiyyətli korrelyasiya əlaqələri müəyyən edilmişdir. 
- Analiz zamanı dataset daxilində potensial outlier (kənar dəyərlər) aşkar edilmişdir.
- Müştərilər balans və tranzaksiya sayına əsasən segmentlərə bölünmüş və bu segmentlər üzrə fərqli davranış modelləri (pattern-lər) müşahidə edilmişdir.
- Anomaliya analizi göstərmişdir ki, bəzi müştərilər ümumi kütlədən kəskin fərqlənən davranış nümayiş etdirir və bu hallar potensial risk və ya xüsusi diqqət tələb edən kateqoriyaya aid edilə bilər.
- Vizual analizlər (boxplot, scatterplot və s.) nəticəsində trend və paylanmalar daha aydın şəkildə ortaya çıxarılmış, xüsusilə anomaly və normal müştərilər arasında fərqlər vizual olaraq təsdiqlənmişdir.
- Ümumilikdə, dataset üzərində aparılan analizlər müştəri davranışını daha yaxşı anlamağa və gələcəkdə risklərin idarə olunması və qərarvermə prosesinin optimallaşdırılmasına imkan yaradır.
- Sadə Machine learning modeli müştərilərin balans, tranzaksiya və digər xüsusiyyətlərinə əsaslanaraq kredit götürmə ehtimalını proqnozlayır.
