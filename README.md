# bank-data-analysis
# Python Data Analiz Layihəsi

## İstifadə olunan alətlər
Pandas,NumPy,Seaborn,Matplotlib
VSCODE-da Exceli bir başa oxumaq üçün ExcelViewer  exstensionu isledile biler
yupter ile ipynb kimi bir sey

## Layihənin təsviri
Bu layihə çərçivəsində verilən dataset üzərində **Exploratory Data Analysis (EDA)** həyata keçirilmişdir.Məlumatlar Python kitabxanaları vasitəsilə yüklənmiş,təmizlənmiş və analiz edilmişdir.Məqsəd dataset-in strukturunu daha yaxşı anlamaq,əsas statistik göstəriciləri əldə etmək və gizli nümunələri üzə çıxarmaq olmuşdur.

Analiz zamanı məlumatların paylanması,dəyişənlər arasındakı əlaqələr və ümumi trend-lər araşdırılmışdır.Vizualizasiya alətləri (Seaborn və Matplotlib) istifadə edilərək müxtəlif qrafiklər qurulmuş və nəticələr daha aydın şəkildə təqdim olunmuşdur.

## Əldə olunan nəticələr (Insights)
- Dəyişənlər arasında müəyyən korrelyasiya əlaqələri aşkar edilmişdir  
- Dataset daxilində potensial outlier-lər  müəyyən edilmişdir  
- Vizual analiz nəticəsində əsas tendensiyalar və pattern-lər üzə çıxarılmışdır
